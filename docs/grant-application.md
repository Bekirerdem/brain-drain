## Step 2 — Details

**Project Details**
> Brain Drain inverts the data economics of AI agents. Today, agents scrape the open web for free; the humans who curate the most valuable knowledge — researchers, engineers, lawyers, doctors — get nothing back. Brain Drain turns any Markdown vault (Obsidian, Notion export, etc.) into an x402-protected micro-API. When an external agent needs niche expert knowledge, it hits a 402 Payment Required, auto-funds a Coinbase CDP Embedded Wallet, sends USDC over Solana mainnet (~400 ms finality via Helius RPC), and unlocks the relevant snippet. Sellers receive payouts in their Phantom Cash balance.
>
> Targeting four Frontier 2026 sponsor bounties simultaneously: Best Multi-Protocol Agent (x402 + AgentPay), Best Use of Phantom CASH, Best Usage of CDP Embedded Wallets, and Best AgentPay Demo. Stack: Next.js 16 + TypeScript on Vercel, Solana mainnet, `@coinbase/cdp-sdk` for buyer-side MPC, x402 HTTP standard for paywalled endpoints, Gemini 3.1 Pro Preview for agent reasoning, `gemini-embedding-001` for RAG, MCP server for Claude Desktop / Cursor integration. Public repo with timestamped commits from Day 0: github.com/Bekirerdem/brain-drain. Solo build, 11-day sprint.
