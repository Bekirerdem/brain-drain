# Why each component is where it is

This is the short version of the decisions baked into Brain Drain. The full reasoning per decision is in the dedicated notes; this is the index.

## Frontend / framework

**Next.js 16 App Router on Vercel.** Tailwind 4 + shadcn for the dashboard. App Router not Pages Router because the agent-facing API routes co-locate cleanly with the seller dashboard and we want React 19 server components in the dashboard.

## Settlement layer

**Solana mainnet, Helius RPC.** Sub-second finality at sub-cent fees is the only viable substrate for $0.05/query economics. See [helius-rpc-low-latency-patterns] for the verification call shape.

## Buyer wallet

**Coinbase CDP Embedded Wallets, not Privy.** Bounty fit + the SDK shape + multisig in pure TypeScript. See [cdp-embedded-wallets-vs-privy].

## Seller payout

**Phantom Cash via SPL transfer to a mainnet Solana address.** The Cash tab aggregates USDC automatically. This is the bounty wedge MCPay/Frames does not target. See [phantom-cash-seller-flow].

## Agent surface

**MCP server with one tool, `brain_drain.query`.** Stdio for Claude Desktop, HTTP for Cursor / Codex. See [mcp-server-architecture-for-solana].

## Reasoning model

**Gemini 3.1 Pro Preview for synthesis, gemini-embedding-001 for RAG, Gemini 2.5 Flash as the cheap fallback.** Claude Haiku 4.5 reserved for the multi-model demo. See [gemini-3-pro-vs-claude-haiku].

## RAG

In-memory cosine similarity over a JSON-on-disk index, regenerated by `scripts/seed-vault.ts` from the `seed-vault/` Markdown directory. No vector DB for v0 — at <500 chunks, a flat scan beats the round-trip latency of any external service. Upgrade to LanceDB or Turbopuffer post-MVP if and when scale demands it.

## What's explicitly NOT here

- No custom Anchor program. See [anchor-free-solana-pattern].
- No multi-tenant seller onboarding for v0 (single seller = me).
- No reputation system for v0 (planned in v1, see [shavaxre-soulbound-roadmap] for the lineage).
- No streaming or subscription model — every query is a one-shot 402.
